# 🔍 Module 05: Exploratory Data Analysis

Welcome to **Module 05: Exploratory Data Analysis (EDA)**. 

Exploratory Data Analysis is the process of performing initial investigations on data to discover patterns, spot anomalies, test hypotheses, and check assumptions using summary statistics and graphical representations.

---

## 🎯 What You Will Learn

In this module, you will learn how to systematically break down and understand any dataset. You will move from basic structural overviews to deep statistical and visual analyses, ensuring you fully comprehend the data before feeding it into a Machine Learning model.

## 🧠 Why is this important?

Models are blind; they only see math. It is the Data Scientist's job to provide the "sight." If you skip EDA, you might build a highly accurate model that solves the wrong problem or learns from hidden biases in the data. Mastering EDA is what separates coders from true Data Scientists.
