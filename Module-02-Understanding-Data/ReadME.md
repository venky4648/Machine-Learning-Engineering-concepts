# 📂 Module 02: Understanding Data

Welcome to **Module 02: Understanding Data**.

Once you have loaded your data into a Pandas DataFrame (as covered in Module 01), the very next step is to understand what you are working with. You cannot clean or model data if you don't know its structure, types, and basic statistics.

This module focuses on the essential Pandas methods used to inspect, preview, and understand a dataset at a high level.
