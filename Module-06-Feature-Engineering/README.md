# 🏗️ Module 06: Feature Engineering

Welcome to **Module 06: Feature Engineering**. 

Feature engineering is the process of using domain knowledge to extract features (characteristics, properties, attributes) from raw data. These features can be used to improve the performance of machine learning algorithms.

---

## 🎯 What You Will Learn

In this module, you will learn how to transform raw, messy data into powerful inputs for your models. You will convert text to numbers, scale varying magnitudes, extract insights from dates, and combine existing columns into entirely new, highly predictive features.

---

## 📂 Topics Covered

Here is the step-by-step roadmap for this module:

1. **[Feature Engineering Introduction](01-Feature-Engineering-Introduction.md)** - What is Feature Engineering and why is it an art?
2. **[Encoding](02-Encoding.md)** - Converting categorical text into numerical format (One-Hot, Label Encoding).
3. **[Scaling](03-Scaling.md)** - Understanding why the magnitude of numbers matters to algorithms.
4. **[Normalization](04-Normalization.md)** - Rescaling data to a range of [0, 1] (Min-Max Scaling).
5. **[Standardization](05-Standardization.md)** - Centering data around a mean of 0 (Z-Score Scaling).
6. **[Binning](06-Binning.md)** - Grouping continuous numbers into categorical buckets.
7. **[DateTime Features](07-DateTime-Features.md)** - Extracting day, month, and year from timestamps.
8. **[Text Features](08-Text-Features.md)** - Basic NLP techniques like extracting word counts and string lengths.
9. **[Feature Creation](09-Feature-Creation.md)** - Using domain knowledge to combine multiple columns.
10. **[Feature Selection](10-Feature-Selection.md)** - Keeping only the most important features to avoid the Curse of Dimensionality.
11. **[Dimensionality Reduction](11-Dimensionality-Reduction.md)** - Using PCA to compress data without losing information.

---

## 🧠 Why is this important?

Andrew Ng famously said: *"Applied machine learning is basically feature engineering."* You can use the most advanced Neural Network in the world, but if you feed it garbage features, you will get garbage predictions. Feature Engineering is how you inject human intelligence into an algorithm.
