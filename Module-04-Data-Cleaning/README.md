# 🧹 Module 04: Data Cleaning

Welcome to **Module 04: Data Cleaning**. 

Now that you have validated your data (found the problems), it is time to clean it (fix the problems). Data cleaning is often cited as taking up 80% of a Data Scientist or Machine Learning Engineer's time. A clean, high-quality dataset is the foundation of any successful ML model.

---

## 🎯 What You Will Learn

In this module, you will learn hands-on techniques to clean dirty data using Pandas. We will cover handling missing values, removing duplicates, treating outliers, standardizing text, and finally, combining all these steps into an automated Data Cleaning Pipeline.

---

## 📂 Topics Covered

Here is the step-by-step roadmap for this module:

1. **[Missing Values](01-Missing-Values.md)** - Learn how to drop or impute missing data (NaNs).
2. **[Duplicate Removal](02-Duplicate-Removal.md)** - Safely remove duplicate records without losing vital information.
3. **[Data Type Conversion](03-Data-Type-Conversion.md)** - Fix incorrect data types so your models can process them.
4. **[Outlier Treatment](04-Outlier-Treatment.md)** - Identify and handle extreme values using statistical methods like Z-Score and IQR.
5. **[Text Cleaning](05-Text-Cleaning.md)** - Standardize strings, remove special characters, and fix casing.
6. **[Inconsistent Data](06-Inconsistent-Data.md)** - Resolve conflicting data entries and map incorrect categories.
7. **[Noise Handling](07-Noise-Handling.md)** - Smooth out random variances in your dataset.
8. **[Data Transformation](08-Data-Transformation.md)** - Apply mathematical transformations to stabilize variance.
9. **[Data Cleaning Pipeline](09-Data-Cleaning-Pipeline.md)** - Combine all cleaning steps into a single, automated, reproducible pipeline.

---

## 🧠 Why is this important?

If you feed messy data into an algorithm, the algorithm will find messy patterns. You cannot out-model bad data. By mastering these data cleaning techniques, you ensure that the insights and models you produce are reliable, robust, and deployable in a real-world production environment.
