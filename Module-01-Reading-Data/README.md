# 📂 Module 01: Reading Data

Welcome to **Module 1: Reading Data**. 

Data is the fuel for any Machine Learning model. Before you can clean data, engineer features, or train models, you must first know how to reliably acquire and load data from various sources into your environment.

In the real world, data doesn't just come in clean `.csv` files. It lives in enterprise SQL databases, is served via REST APIs, is stored in massively compressed Parquet files in Data Lakes, and is distributed across Cloud Storage buckets.

This module covers all the essential file formats and data sources a Machine Learning Engineer must master.
